# README

Technical Requirements:
1. Ruby 2.5.1
2. Rails 5.2.1
3. Posgresql 9.6.x

Do 'bundle install' before starting up rails server

Gem library used
1. public_suffix
2. geoip
3. jquery
4. bootstrap

Usage:
http://localhost:3000/api/v1/www.google.com
To get short url

http://localhost:3000/api/v1/details/www.google.com
To get detail of short url

http://localhost:3000
To see summary page
